# Web Templates

## Concept
Ce site regroupe mes références personnelles de designs de sites internet.  
Il contient également une liste de références et d'outils pour aider à la création de sites.

<br>

## Technologies
- Front-end : Vue.js

### Utilisation
- `npm install` : Installe les dépendances
- `launch.bat` : Lance l'application avec vite

### Procédure de commit
- `git commit -m message`
- `node dev\incrementVersion.js` : Incrémente APP_VERSION.js + git tag version
- `dev\gitPush.bat` : Push le commit et les tags
